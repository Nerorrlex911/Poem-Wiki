# Dependencies

:::info

Minestom需要Java 17或更高版本才能运行。 如果您使用的是Gradle，则必须使用7.2或更高版本。

:::

将Minestom添加到您的Java项目中非常简单，您只需要添加几个存储库：

## Repositories

### Gradle (Groovy)

```groovy
repositories {
    // ...
    mavenCentral()
    maven { url 'https://jitpack.io' }
}
```

### Gradle (Kotlin)

```groovy
repositories {
    // ...
    mavenCentral()
    maven(url = "https://jitpack.io")
}
```


### Maven

```markup
<repositories>
    <!-- ... -->
    <repository>
        <id>jitpack</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```


## Dependencies

### Gradle (Groovy)

```groovy
dependencies {
    // ...
    implementation 'com.github.Minestom:Minestom:VERSION'
}
```
### Gradle (Kotlin)

```groovy
dependencies {
    //...
    implementation("com.github.Minestom:Minestom:VERSION")
}
```

### Maven

```markup
<dependencies>
    <!-- ... -->
    <dependency>
        <groupId>com.github.Minestom</groupId>
        <artifactId>Minestom</artifactId>
        <version>VERSION</version>
        <exclusions>
            <exclusion>
                <groupId>org.jboss.shrinkwrap.resolver</groupId>
                <artifactId>shrinkwrap-resolver-depchain</artifactId>
            </exclusion>
        </exclusions>
    </dependency>
</dependencies>
```

当使用Maven时，建议从组`org.jboss.shrinkwrap.resolver`中排除`shrinkwrap-resolver-depchain`，否则将无法解析依赖项。 如果需要，可以单独添加Shrinkwrap作为依赖项以解决问题。

版本列表可以在 [https://jitpack.io/#Minestom/Minestom](https://jitpack.io/#Minestom/Minestom)找到。
